How to Add Your Hero Image

To add your own hero image to Potluck:

  1. Create a folder named images in your project directory
  2. Add your hero image to the folder (name it potluck-hero.jpg)
  3. Make sure the image is at least 1200x400 pixels for best quality
  4. Supported formats: JPG, PNG, or WebP

Example image sources: food photography, cooking scene, ingredients collage, or kitchen setting
